🚀 MongoDB Admin
O gerenciador de MongoDB nativo, escrito em Rust, focado em performance extrema, baixo consumo de memória e conectividade sem fricção.

Diferente de outros gerenciadores baseados em Electron, o MongoDB Admin é leve, rápido e projetado para desenvolvedores que precisam de produtividade real, especialmente em ambientes de nuvem como AWS DocumentDB.

✨ Principais Diferenciais
🚀 Performance Rust Nativa: Consumo de RAM até 10x menor que o MongoDB Compass.

☁️ Conectividade Cloud Real: Conexão nativa e estável com AWS DocumentDB e Azure CosmosDB (API Mongo).

📂 Organização por Ambientes: Separação lógica de conexões por projetos ou ambientes (Dev, Staging, Prod) em uma interface organizada.

🖱️ Smart Filtering: Clique em qualquer chave (Key) na listagem de coleções e filtre instantaneamente por valor.

🛠️ Universal Client: Compatível com qualquer versão do MongoDB e drivers existentes.
